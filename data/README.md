# Data Directory

This directory holds all data artefacts for the vlm_video project.
**None of the subdirectories (except this README) are tracked by git.**

## Directory structure

```
data/
├── raw/          ← Original, immutable source videos (MP4, MKV, …)
│   └── <video_id>.mp4
├── interim/      ← Intermediate outputs generated by the pipeline scripts
│   └── <video_id>/
│       ├── frames/            ← JPEG frames (01_extract_frames.py)
│       │   ├── frame_000001.jpg
│       │   └── frame_manifest.jsonl
│       ├── audio.wav          ← Extracted mono 16 kHz audio
│       ├── transcript.jsonl   ← ASR output (02_run_asr.py)
│       ├── ocr_results.jsonl  ← Optional OCR output (03_run_ocr.py)
│       ├── embeddings.npz     ← Fused per-frame embeddings (04_build_embeddings.py)
│       ├── embedding_meta.jsonl
│       ├── segments_pred.jsonl ← Segmentation output (05_segment_video.py)
│       └── index/             ← Retrieval index files (06_build_index.py)
└── processed/    ← Final, analysis-ready datasets and ground-truth annotations
    └── <dataset_name>/
        ├── annotations.jsonl  ← Manual boundary annotations
        └── queries.jsonl      ← Evaluation queries with relevance labels
```

## Expected video formats

| Property  | Requirement                    |
|-----------|--------------------------------|
| Container | MP4, MKV, AVI, MOV, WEBM       |
| Video     | H.264 or H.265 recommended     |
| Audio     | Any codec (re-encoded to WAV)  |
| Duration  | ≤ 2 hours per file             |
| Naming    | Use ASCII `video_id` (no spaces) |

## Ground-truth annotation format (`annotations.jsonl`)

Each line is a JSON object:
```json
{"video_id": "lec01", "boundaries_sec": [120.0, 340.5, 600.0], "annotator": "A1"}
```

## Licensing notes

- Place only data you have the right to use for research purposes.
- Do **not** commit copyrighted lecture videos to this repository.
- If using public datasets, document the license in `data/processed/<dataset>/LICENSE`.
- For internal university data, refer to the institutional data-use agreement.
